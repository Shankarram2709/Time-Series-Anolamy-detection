# Time-Series-Anomaly-detection-and using RNN-Python
Implementation of Recurrent Neural Network based anomaly detection in a Univariate time series in Tensorflow.

# Requirements

* WINDOWS 10
* Python 3.5+
* Tensorflow 2.0
* Tensorflow Estimator<2.1
* Numpy
* Matplotlib
* Pandas
* Seaborn
